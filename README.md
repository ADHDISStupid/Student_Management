# Student Management System (Python)

A simple command-line Student Management System built with Python.  
It allows you to add, remove, view, and search students using a text file for storage.

---

## Features

- Add students  
- Remove students  
- View all students  
- Search for a student  
- Saves data to `Student.txt` automatically  
- Loads data when the program starts  

---
